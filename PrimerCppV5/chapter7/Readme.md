###练习7.8 
>为什么read函数将其Sales_data 参数定义成普通的引用，而print 将其参数定义成常量引用？
```
定义成普通的引用，表明我们可以去修改参数的值，定义成常量引用表示我们不可以修改参数的值，仅可以去读它，而不可以去写这个值。
```
